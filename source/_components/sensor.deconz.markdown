See the [deCONZ main component](/components/deconz/) for configuration instructions.

The following sensor types are supported:

 * Humidity sensor
 * Light level sensor
 * Pressure sensor
 * Switches
 * Temperature sensor

Entity ids will be sensor.device_name, where device_name comes from deCONZ.
